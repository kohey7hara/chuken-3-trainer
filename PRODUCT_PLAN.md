# 中国語学習アプリ計画

## 目的

毎日20分の学習で中国語の基礎を固め、中国語検定3級合格を最初の到達点にする。
その後、中国語検定2級に必要な読解、翻訳、作文、リスニング力へ拡張する。

## 最初に作るもの

MVPは「中検3級合格までの毎日学習Webアプリ」とする。
スマホとPCの両方で使えるサイトとして作り、アプリ化は後回しにする。

## MVP機能

1. 今日の学習
   - 単語5問
   - 文法3問
   - リスニング1問
   - 短い発音練習1問

2. 単語カード
   - 中国語
   - ピンイン
   - 日本語
   - 例文
   - 習熟度

3. 文法ドリル
   - 語順並べ替え
   - 穴埋め
   - 日文中訳
   - 文法ポイント解説

4. リスニング
   - 音声再生
   - 内容一致問題
   - ディクテーション

5. スピーキング
   - 例文の音読
   - 録音
   - 自己評価
   - 将来的にAI発音評価を追加

6. 進捗管理
   - 連続学習日数
   - 3級合格までの進捗
   - 弱点カテゴリ

## 学習ロードマップ

### Phase 1: 発音と基礎語順

期間: 1か月

- ピンイン
- 声調
- 基本文型
- 疑問文
- 否定文
- 基本単語500語

### Phase 2: 3級基礎文法

期間: 2か月

- 了
- 过
- 着
- 比較文
- 可能補語
- 結果補語
- 方向補語
- 連動文
- 兼語文
- 基本単語1,000語

### Phase 3: 3級対策

期間: 3か月

- 常用語2,000語
- リスニング演習
- 長文読解
- 日文中訳
- 過去問形式の模試

### Phase 4: 2級準備

3級合格後に開始する。

- 複文
- 慣用句
- 長文読解
- 中日訳
- 日中訳
- 中国語作文
- 社会生活テーマのリスニング

## 最初の画面

最初の画面はランディングページではなく、すぐ学習に入れるダッシュボードにする。

- 今日のミッション
- 学習開始ボタン
- 3級までの進捗
- 今日の復習カード
- 弱点カテゴリ

## コンテンツ方針

最初は少量でもよいので、以下の形式で教材データを持つ。

- lesson
- vocabulary
- grammar
- listening
- speaking
- quiz

教材はアプリ本体と分離し、後から増やせるようにする。

## 技術方針

最初はローカルで動くWebアプリとして作る。

- フロントエンド: React または Next.js
- データ: JSON または SQLite
- 音声: ブラウザの音声合成から開始
- 学習記録: localStorage から開始

ユーザー登録、クラウド同期、AI添削は初期版では入れない。
毎日使える最小体験を先に完成させる。
